# Best Burgers Website - Flask-Blog-Assessment-Task-1
A clean, interactive and highly accessible cooking blog website featuring various different delicious burger recipes, built with the Python Flask library, HTML, CSS and JavaScript. Coded entirely by Kaleb Vong, Documentation by Allan Luong and Presentation by Danny T. Nguyen. Coding

## Features
- Extremely easy and simple start procedure with localhost
- Modest and clean layout with two separate pages for recipes and questions
- Tags and sidebar for easy filtering and finding specific posts
- Ability to post comments under discussions and recipe pages
- Admin privileges for trusted users capable of editing and deleting posts by users

## Installation Guide
### Prerequisites
- Python (version 3.x or higher) and PIP (version 25.x or higher)

### Step-by-Step Installation
1. Click the green "Code" button on GitHub page and click "Download ZIP"
2. Navigate to "Flask-Website-Assessment-Task-1-main.zip" and open it by **Extracting** or third-party software such as [WinRAR](https://www.win-rar.com/download.html?&L=0)
3. Drag out the **Flask-Website-Task-1-main folder** from inside the ZIP file or from the folder created by **extracting** (if extraction method was used) to any suitable location of your choosing
4. Rename **env.default file** in **Flask_Blog folder** to **.env**
5. Double-click **run.bat file** to kickstart website locally
6. **Ctrl + L-Click** website URL to access website
